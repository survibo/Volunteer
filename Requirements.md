# 요구사항 문서

## 1. 개요

Google OAuth 또는 Kakao OAuth 기반 로그인을 사용하는 봉사활동 및 교육 신청 관리 시스템이다.
준회원, 회원, 관리자로 역할이 구분되며, OAuth 로그인 후 관리자의 수동 승인을 거쳐 준회원에서 회원으로 전환된다.

---

## 2. 사용자 역할

| 역할 | 설명 |
|------|------|
| 준회원 | Google OAuth 또는 Kakao Oauth 로그인 후 관리자 수락 전. 회원번호 없음. 기능은 회원과 동일 |
| 회원 | 관리자 수락 완료. 회원번호 부여됨 |
| 관리자 | 준회원 승인, 봉사활동/교육 관리 전반을 담당 |

---

## 3. 인증 및 가입

### 3.1 로그인

- Google OAuth 2.0 또는 Kakao OAuth 2.0을 통해 로그인한다.
- 최초 로그인 시 준회원으로 등록된다.

### 3.2 가입 정보 입력

- Google OAuth 또는 Kakao OAuth 최초 로그인 후, 준회원 등록 전에 가입 정보 입력 단계를 거친다.
- 아래 항목을 본인이 직접 입력해야 하며, 모두 필수이다.

| 항목 | 비고 |
|------|------|
| 이름 | 필수 |
| 연락처 | 필수 |
| 이메일 | 필수 |
| 주소 | 필수 |
| 근무지 or 학교명 | 필수 |
| 면허번호 | 선택 |


- 입력 완료 후 준회원으로 등록된다.

### 3.3 회원 전환 흐름

1. 사용자가 Google OAuth 또는 Kakao OAuth로 최초 로그인한다.
2. 가입 정보를 입력한다.
3. 준회원으로 등록된다.
4. 사용자는 가입비를 별도 계좌로 입금한다.
5. 관리자가 외부에서 입금을 확인한 후 시스템에서 수동으로 수락한다.
6. 수락 시 회원번호가 자동 부여되고 회원으로 전환된다.

### 3.4 회원번호 규칙

- 형식: `YY-NNNN`
- `YY`: 회원 전환 연도 2자리
- `NNNN`: 해당 연도 내 전환 순서 (4자리, 0001부터 순차 증가)
- 예시: `26-0001`, `26-0002`

### 3.5 회원 탈퇴

- 준회원 및 회원은 마이페이지에서 직접 탈퇴를 신청할 수 있다.
- 탈퇴 시 아래 정보를 시스템에 보관한다 (관리자 열람 목적).

| 보관 항목 | 비고 |
|-----------|------|
| 이름 | |
| 연락처 | |
| 이메일 | |
| 회원번호 | 준회원의 경우 없음 |
| 탈퇴 일시 | 자동 기록 |
| 탈퇴 시 역할 | 준회원 / 회원 구분 |

- 탈퇴 처리 후 해당 계정으로는 로그인할 수 없다.
- 탈퇴한 사용자가 동일 OAuth 계정으로 재가입할 경우, 신규 준회원으로 처음부터 절차를 밟는다.

## 4. 봉사활동 신청

### 4.1 관리자 기능

- 봉사활동을 개설, 수정, 폐지할 수 있다.
- 개설 시 제목, 설명(선택), 이미지(선택), 장소, 모집 기한, 모집 인원(n명)을 설정한다.
- 기한 중 신청을 실시간으로 수락 또는 거절할 수 있다.
- 정원이 초과되어도 신청은 계속 받는다. 초과 인원 조절은 관리자가 직접 한다.
- 기한 종료 후에는 관리자만 수락된 인원을 취소할 수 있다.

### 4.2 준회원/회원 기능

- 모집 기한 내에 봉사활동을 신청할 수 있다.
- 기한 중에는 본인의 신청을 취소할 수 있다.
- 기한 종료 후에는 취소할 수 없다.

### 4.3 신청 상태 흐름

```
신청 대기
  ├── 수락 → 수락됨
  ├── 거절 → 거절됨
  └── 본인 취소 (기한 중) → 취소됨

수락됨
  └── 관리자 취소 (기한 후 포함) → 취소됨
```

---

## 5. 교육 신청

- 봉사활동 신청과 구조가 동일하다.
- 명칭만 "봉사활동" 대신 "교육"으로 다르다.
- 관리자 기능, 준회원/회원 기능, 신청 상태 흐름 모두 4항과 동일하게 적용된다.

---

## 6. 마이페이지

### 6.1 봉사활동 내역

- 현재 신청 중인 봉사활동 목록 (신청 대기, 수락됨 상태)
- 지난 봉사활동 내역 (수락됨, 거절됨, 취소됨 포함)

### 6.2 교육 내역

- 현재 신청 중인 교육 목록
- 지난 교육 내역

### 6.3 회원 탈퇴

- 마이페이지 하단에서 탈퇴를 신청할 수 있다.
- 탈퇴 전 의사를 재확인하는 안내 문구 및 확인 절차를 거친다.

| 기능 | 설명 |
